Installation
============

spann requires Python ``>=3.8`` and can be installed from pypi via:

.. code-block:: bash

   python -m pip install spann


To install directly from github, you can run:

.. code-block:: bash

   python -m pip install git+https://github.com/michaelGuo1204/SPANN-torch.git

For development, its recommended to use Poetry:

.. code-block:: bash

   git clone https://github.com/michaelGuo1204/SPANN-torch.git
   cd spann
   poetry install
